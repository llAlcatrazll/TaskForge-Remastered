# 🧠 Personal Compendium App

An **all-in-one modular desktop application** that centralizes:

- personal tracking
- productivity tools
- system monitoring

Built with **WinUI 3 and .NET 8**.

---

# ✨ Features

## 🟢 Core Modules
- 📝 To-Do List
- 💰 Money Tracker
- 📚 Manhwa Tracker
- 📺 Anime Tracker
- 🚀 Quick Home Launcher
- 🎮 Steam Tracker
- 🕒 PC Uptime Tracker

## 🟡 Productivity Tools
- 🖼️ Background Remover
- 🎯 Work Focus Mode
- ⏱️ App Timeout
- 💳 Subscription Manager
- 📊 PC App Usage Tracker
- 🔔 Daily Reminder
- 💻 Code Snippets

## 🔴 System Intelligence
- 🧹 Storage Discoverer
- 🤖 Auto-Cleanup Suggestions
- 🔐 File Vault
- 🌐 Universal Notification Collection
- 👥 Presence Detector
- 📋 Clipboard Manager
- 🧩 Custom Desktop Widgets
- 🐙 GitHub Tracker
